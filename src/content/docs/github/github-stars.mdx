## Sizes

The GitHub Stars component supports different sizes:
```tsx
<GitHubStars size="lg" />
<GitHubStars size="default" />
<GitHubStars size="sm" />
```

### Default Size

<GitHubStars 
  repo="block/goose"/>

### Large

<GitHubStars repo="block/goose" size="lg" />

### Small 
<GitHubStars repo="block/goose" size="sm" />

## Properties

The GitHub Stars component accepts the following props:

| Prop      | Type                                                                                 | Default     | Description                               |
|-----------|--------------------------------------------------------------------------------------|-------------|-------------------------------------------|
| size      | `"default" | "sm" | "lg" | "icon"`                                               | `"default"` | The size of the button                    |
| repo   | `string`                                                                            | None     | Required; the name of the repository to get GitHub stars for, e.g. "block/goose"    |
