# FeatureCard

A flexible card layout for feature sections, with left and right panes. Accepts a label, headline, description, and custom content for each pane.

## Example

<FeatureCard
  label="Step 1"
  headline="Quick Installation"
  description="Get started in seconds with our pre-configured template. Just clone, install, and start building."
>
  ```git clone https://github.com/block/docs-site-kickstarter.git my-docs
    cd my-docs
    pnpm install
    pnpm dev
  ```
  { <div slot="content">
    Right side content here.
  </div> }
</FeatureCard>

## Usage

```tsx
import FeatureCard from "@/components/block/FeatureCard.astro"

<FeatureCard
  label="Step 1"
  headline="Quick Installation"
  description="Get started in seconds with our pre-configured template. Just clone, install, and start building."
>
  <pre className="language-bash rounded-lg p-4 bg-card/40 backdrop-blur-sm overflow-x-auto">
    git clone https://github.com/block/docs-site-kickstarter.git my-docs
    cd my-docs
    pnpm install
    pnpm dev
  </pre>
  <div slot="content">
    Right side content here.
  </div>
</FeatureCard>
```
