## Customization

This template is designed to be easily customizable. You can:

- **Modify the theme** using Tailwind CSS variables
- **Add new components** from shadcn/ui or create your own
- **Create custom documentation components** for specific use cases
- **Extend the MDX configuration** to add new features

Check out our [configuration guide](/docs-site-kickstarter/docs/configuration) for detailed customization instructions.
