## Introduction

Welcome to the documentation. This is a template for building documentation sites using Astro and shadcn/ui.

## Key Features

- **Built with Astro** - Fast, modern static site generator
- **Styled with Tailwind CSS** - Utility-first CSS framework
- **UI components from shadcn/ui** - High-quality, accessible components
- **Dark mode support** - Automatic and manual dark mode
- **Full-text search** (coming soon) - Easy content discovery
- **Documentation-specific components** - Purpose-built for documentation
- **MDX support** - Use React components in Markdown
- **TypeScript** - Full type safety and autocompletion

## Getting Started

## Customization

This template is designed to be easily customizable. You can:

- Modify the theme using Tailwind CSS
- Add new components from shadcn/ui
- Create custom documentation components
- Extend the MDX configuration

Check out our [configuration guide](/docs/configuration) for more details.
