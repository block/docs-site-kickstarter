# MDX Example

This is an example documentation page written in MDX. MDX allows you to use JSX components directly in your markdown content.

## Interactive Components

Here's an example of using a React component in MDX:

<Button>Click me!</Button>

## Code Examples

```tsx
// You can include code blocks with syntax highlighting
import { Button } from "@/components/ui/button";

export default function Example() {
  return (
    <Button variant="outline">
      Click me
    </Button>
  );
}
```

## Lists and Typography

- Including lists
- And other formatting

## Custom Components

You can import and use any React component:

<div className="bg-muted p-4 rounded-lg">
  This is a custom styled div using Tailwind classes
</div>

MDX can also support math equations if you add the right plugins:

```math
E = mc^2
```
