# CallToAction

A prominent call-to-action section with headline, description, and one or two buttons.

## Example

<CallToAction
  headline="Ready to Build Your Documentation?"
  description="Start creating beautiful documentation sites for your Block projects today."
  primaryCTA="Get Started"
  primaryCTALink="/docs-site-kickstarter/docs"
  secondaryCTA="View on GitHub"
  secondaryCTALink="https://github.com/block/docs-site-kickstarter"
/>
